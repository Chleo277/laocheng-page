禁用 Windows10 系统自动更新

IT 技术 Windows10 作者：老成
1
禁用 Windows Update 服务
电脑 → 右键 → 管理 → 服务和应用程序 → 点击服务 → 找到 Windows Update
双击 → 启动类型选 禁用 → 如果运行中，点 停止
点击 应用、确定
切换到「恢复」标签 → 第一次失败选 无操作 → 计数填 9999天
点击 确定
2
禁用 Microsoft Edge 自动更新任务
管理界面 → 系统工具 → 任务计划程序 → 任务计划程序库
找到两个 Microsoft Edge Update
分别右键 → 选择 禁用
3
禁用 Windows Update 计划任务
计算机管理界面 → 系统工具 → 任务计划程序 → 任务计划程序库 → Microsoft → Windows
下拉找到 Windows Update
右侧选中 Scheduled Start → 右键 → 选择 禁用
